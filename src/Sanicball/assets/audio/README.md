# Folder Structure

## Ambience
* Audio that is meant to be played alongside music
* e.g. sea, birds

## Music
* Audio that is meant to be the main source of sound
* Controlled by the players
* e.g. Sonic OSTs

## SFX
* Sound effects
* e.g. Speed up, ring got

## Song
* The type of a Song
* Contains the audio file, composer and other metadata
